Various OS settings to toggle on new systems.

# Settings

-   Input Sources
    -   turn off spelling autocorrect
        -   turn off auto caps
        -   turn off inline predictive text
-   Trackpad
    -   scroll and zoom: change scroll direction
-   Keyboard Shortcuts
    -   Spotlight --> turn off spotlight search and open finder window
-   Keyboard
    -   press global key to do nothing
    -   key repeat rate: fast
    -   delay until repeat: second-last one
-   Desktop & Dock
    -   default web browser: chrome
    -   minimize windows into application icon: on
    -   show suggested and recent apps in dock: off
-   Control Center
    -   battery —> show percentage
    -   display the time with seconds in menu bar
-   Screen Saver
    -   shuffle all
-   cmd shift 5: save screenshots to clipboard
-   Sound --> play user interface sound effects off, alert volume 0
