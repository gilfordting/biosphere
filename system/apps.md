Apps to install on a new system.

-   [Google Chrome](https://www.google.com/chrome/)

# System utilities

-   [Raycast](https://www.raycast.com/download/)
    -   Change shortcut to Cmd + Space
    -   Emoji picker: Option + Cmd + Space
-   [Rectangle](https://rectangleapp.com/)
    -   launch on login
    -   check for updates automatically
-   [Jiggler](https://www.sticksoftware.com/software/Jiggler.html)
-   [AltTab](https://alt-tab-macos.netlify.app/)
    -   change shortcut to Cmd + Tab

# Personal apps

-   [Spotify](https://www.spotify.com/us/download/other/)

# Productivity

-   [Obsidian](https://obsidian.md/download)
    -   Plugins:
        -   Git
        -   Latex Suite
            -   some of these snippets are annoying
        -   Smart Typography (for em dashes)
            -   also some annoying snippets here
        -   Paste URL into selection
        -   Paste image rename
    -   Hotkeys
        -   Toggle left/right sidebar with Cmd + \[, \]
-   [Todoist](https://www.todoist.com/downloads)
-   [Slack](https://slack.com/downloads/mac)
    -   preferences --> messages and media --> emoji --> convert typed.. off
-   [Zoom](https://zoom.us/download)

# Coding

-   [iTerm2](https://iterm2.com/) with settings:
    -   iterm2 --> install shell integration
    -   Profiles
        -   Terminal
            -   unlimited scrollback
            -   show mark indicators off
        -   keys
            -   key bindings —> presets —> natural text editing
        -   colors
            -   edit light mode
            -   defaults: background 1c1f24, foreground dbdbdb
            -   cursor: background fefefe, foreground 000000
            -   use custom color bold text: fefefe
            -   background: something slightly lighter
        -   window
            -   adjust default window size
    -   keys
        -   hotkey --> show/hide all windows with dedicated hotkey
    -   appearance --> general --> theme: minimal
-   [Homebrew](https://brew.sh/)
-   [Cursor](https://cursor.com/download)
    -   Install 'cursor' command
